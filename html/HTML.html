JS HTML DOM: cambio de HTML
 
El DOM HTML permite que JavaScript cambie el contenido de los elementos HTML.
Cambio de contenido HTML
 

La forma más sencilla de modificar el contenido de un elemento HTML es mediante la innerHTML propiedad.

Para cambiar el contenido de un elemento HTML, use esta sintaxis:

document.getElementById(id).innerHTML = new HTML

Este ejemplo cambia el contenido de un < p > elemento:

Ejemplo
< html>
< body>

< h2>JavaScript puede cambiar HTML< /h2>
< p id="p1">Hello World!< /p>

< script>
document.getElementById("p1").innerHTML = "New text!";
< /script>

< p >El párrafo anterior fue cambiado por el script.< /p>
< /body>
< /html>

Esta es la manera en que podemos visualizar nuestro ejemplo al ejecutarlo


Cambiar el valor de un atributo HTML
 

Para cambiar el valor de un atributo HTML, use esta sintaxis:

document.getElementById(id).attribute = new value

Este ejemplo cambia el valor del atributo src de un < img> elemento:

Ejemplo
< html>
< body>

< img id="myImage" src="../img/smiley.jpg">
< script>
document.getElementById("myImage").src = "../img/landscape.jpg";
< /script>

< p>La imagen original era smiley.jpg, pero el script la cambió a landscape.jpg< /p>
< /body>
< /html>

Esta es la manera en que podemos visualizar nuestro ejemplo al ejecutarlo


Contenido HTML dinámico
 

JavaScript puede crear contenido HTML dinámico:

Ejemplo
< !DOCTYPE html>>
< body>

< script>
document.getElementById("demo").innerHTML = "Date : " + Date();
< /script>

< /body>
< /html>

Esta es la manera en que podemos visualizar nuestro ejemplo al ejecutarlo


document.write ()
 

En JavaScript, document.write() se puede utilizar para escribir directamente en el flujo de salida HTML:

Ejemplo
< html>
< body>

< p>Bla bla bla< /p>
< script>
document.write(Date());
< /script>

< p>Bla bla bla< /p>
< /body>
< /html>

Esta es la manera en que podemos visualizar nuestro ejemplo al ejecutarlo
