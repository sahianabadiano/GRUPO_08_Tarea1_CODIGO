JS HTML DOM: métodos HTML DOM
 
Se trata de acciones que se pueden realizar en elementos HTML. y sus propiedades son valores que se pueden cambiar.

La interfaz de programación DOM


  


Son las propiedades y métodos de cada objeto.


  • Se puede acceder al DOM HTML con JavaScript y otros lenguajes de programación.
  • En el DOM todos los elementos HTML se definen como objetos.
  • Una propiedad es un valor que se puede obtener o establecer.
  • Un método es una acción que se puede realizar.
