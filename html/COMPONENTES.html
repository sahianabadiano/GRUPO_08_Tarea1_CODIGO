    Validación de formulario JavaScript
     

    function validateForm() {
    let x = document.forms["myForm"]["fname"].value;
    if (x == "") {
    alert("Name must be filled out");
    return false;
    }
    }

    La funcion se llama cuando se envie el formulario

    Ejemplo

    Ejemplo
    < !DOCTYPE html>
    < html>
    < head>

    < script>
    function validateForm() {
    let x = document.forms["myForm"]["fname"].value;
    if (x == "") {
    alert("El nombre debe llenarse");
    return false;
    }
    }
    < /script>

    < /head>
    < body>
    < h2>JavaScript Validation< /h2>

    < form name="myForm" action="/action_page.php" onsubmit="return validateForm()" method="post">
    Name: < input type="text" name="fname">
    < input type="submit" value="Enviar">
    < /form>

    < /body>
    < /html>

    Esta es la manera en que podemos visualizar nuestro ejemplo al ejecutarlo

    Encontrar elementos HTML


      


    Respecto a Documentos, DOM puede trabajar con los componentes principales que son:


    Metodo

    Descripcion
    document.getElementById(id) Encontrar un elemento por su ID
    document.getElementsByTagName(name) Encontrar un elemento por nombre de etiqueta
    document.getElementsByClassName(name) Encontrar un elemento por nombre de clase


    Cambiar elementos HTML


      



    Propiedad

    Descripcion
    element.innerHTML = new html content Cambiar el HTML interno de un elemento
    element.attribute = new value Cambiar el valor del atributo de un elemento HTML
    element.style.property = new style Cambiar el estilo de un elemento HTML


    Agregar y eliminar elementos HTML


      



    Metodo

    Descripcion
    document.createElement(element) Crear un nuevo elemento HTML
    document.removeChild(element) Remover un elemento HTML
    document.appendChild(element) Añadir y remplazar un elemento HTML
    document.replaceChild(new, old) Remplazar un elemento HTML
    document.write(text) Escribir en el flujo de salida HTML
