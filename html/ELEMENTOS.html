JS HTML DOM: Elementos HTML DOM
 
A menudo, con JavaScript, desea manipular elementos HTML.

Elementos DOM HTML


  


Para hacerlo, primero debes encontrar los elementos. Hay varias maneras de hacer esto:


  • Encontrar elementos HTML por ID
  • Encontrar elementos HTML por nombre de etiqueta
  • Encontrar elementos HTML por nombre de clase
  • Encontrar elementos HTML mediante selectores CSS
  • Encontrar elementos HTML por colecciones de objetos HTML


    • Leer Más
    Encontrar elemento HTML por ID
     

    const element = document.getElementById("intro");

    Si se encuentra el elemento, el método devolverá el elemento como un objeto (en myElement).

    Si no se encuentra el elemento, myElement contendrá null.

    Ejemplo

    Ejemplo
    < !DOCTYPE html>
    < html>
    < body>
    < h2>JavaScript HTML DOM< /h2>
    < p id="intro">Búsqueda de elementos HTML por ID< /p>
    < p>Este ejemplo demuestra el método < b>getElementsById< /b>< /p>
    < p id="demo">< /p>
    < script> const element = document.getElementById("intro");
    document.getElementById("demo").innerHTML =
    "El texto del párrafo de introducción es: " + element.innerHTML;
    < /script>
    < /body>
    < /html>

    Esta es la manera en que podemos visualizar nuestro ejemplo al ejecutarlo


    Encontrar elementos HTML por nombre de etiqueta
     


    const element = document.getElementsByTagName("p");

    Muestra todos los p elementos

    Ejemplo

    Ejemplo
    < !DOCTYPE html>
    < html>
    < body>

    < h2>JavaScript HTML DOM< /h2>

    < p>Finding HTML Elements by Tag Name.< /p>
    < p>This example demonstrates the < b>getElementsByTagName< /b> method.< /p>

    < p id="demo">< /p>

    < script>
    const element = document.getElementsByTagName("p");
    document.getElementById("demo").innerHTML = 'The text in first paragraph (index 0) is: ' + element[0].innerHTML;
    < /script>

    < /body>
    < /html>

    Esta es la manera en que podemos visualizar nuestro ejemplo al ejecutarlo

    Encontrar elementos HTML por nombre de clase
     


    const x = document.getElementsByClassName("intro");
